Counties in State
This application displays a manually-sortable list of counties in two different states. It makes use of Python (v2.7), web.py (v3.2), SqlAlchemy(v0.6.8), Mako (v0.3.6), Mootools (v1.4.0.1), and JSON. The application was developed with Apache serving the content via mod_wsgi. It is designed to work with Firefox 7. Source code is available as a tar.gz file (here), or from a github repository (here).
